## 🏠 Configuration des Appareils

### Caméras
1. Positionnement optimal
   - Hauteur recommandée : 2.1m - 2.4m
   - Angle de vue : 30-45 degrés
   - Évitez la lumière directe
   
2. Configuration réseau
   - Connexion Wi-Fi sécurisée
   - Bande passante minimale : 2Mbps
   - Port requis : 8000-8999

### Serrures Connectées
1. Installation

## 🚨 Gestion des Alertes

### Types d'Alertes
1. Intrusion
   - Détection de mouvement
   - Ouverture non autorisée
   - Tentative d'accès échouée

2. Technique
   - Batterie faible
   - Perte de connexion
   - Dysfonctionnement

### Procédures d'Urgence
1. Intrusion Détectée
   - Vérifiez les caméras
   - Contactez la sécurité
   - Activez la sirène

2. Fausse Alerte
   - Désactivation rapide
   - Vérification système
   - Ajustement paramètres
